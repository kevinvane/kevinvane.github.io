Win10降级Win7知识

发表于 2019-08-23 更新于 2023-08-09 分类于 随笔

Win8、Win10对技术更新了不少，现在的购买的机器都是预装Win10，如果想要降级为Win7，那么如果想简单的重装系统，会发现装不了。

原因

  1. 新的机器都是采用 UEFI BIOS 启动方式 ，它具有启动速度快、安全性高和支持大容量硬盘的特点。
  2. Win10的硬盘分区采用 GUID 格式

修改

如果要降级重装 Win7，修改这两个东西：

  1. 设置 BIOS 为 Legacy 启动方式，这种是老机器用的方式。
  2. 将硬盘分区转换为MBR格式（利用PE分区工具）

BIOS具体操作

  1. 进入 BIOS 中的 Boot 项
  2. Secure Boot 关闭掉（disable）
  3. Boot List Option 选择 Legacy

硬盘分区操作

  1. 利用PE的分区工具
  2. 找到 转换分区表类型为MBR格式 的功能

这样应该就能装Win7了。
