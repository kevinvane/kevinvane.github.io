FastReport.Net安装

发表于 2023-05-01 更新于 2023-05-05 分类于 FastReport

项目需求：由自己的程序根据数据打印二维码标签，虽然调用 Win32 API 可以实现简单的格式，但利用 FastReport 报表模板显然是一个更好的解决方案，可维护性更好。

FastReport的功能很强大，报表模板打印只是它的一个小功能。

一、前言

FastReport.Net —— 报表开发.NET动态库，适用于 C# 编程语言，报表模板文件是frx。

FastReport.Net 使用逻辑是：

  1. 用Designer.exe设计模板;
  2. 项目引用DLL动态链接库;
  3. 调用 API 设置标签变量参数、打印机属性
  4. 打印

相比 VCL版本，FastReport.Net的安装和配置相对简单了许多。

6. 项目通过Nuget安装引入DLL

“Nuget包管理器” 上可以安装 FastReport 引用，“Nuget包管理器” 会自动安装 FastReport.dll 及它所需要的依赖项。

总结

FastReport 是一个功能很齐全的报表工具，本文只是涉及到利用它的库打印报表。安装了FRNetDemo.msi，可以体验它最完整的功能，比如：

  1. 可视化定制报表；
  2. 支持导出PDF、HTML等内容；
  3. 支持字符串、XML、JSON、各类数据库作为数据源；
  4. 支持脚本自动生成报表
  5. 等等。。。
