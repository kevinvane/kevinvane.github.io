描述

multipart/form-data 用以向服务器发送二进制数据，一般用于多个文件和参数一起发送。

  1. 每一个文件或者参数做为一个Part
  2. Part 都包含头信息部分
  3. Part 头信息中必须包含一个 Content-Disposition 头，其他的头信息则为可选项， 比如 Content-Type 等
  4. 每个 Part 使用 --boundary 分割，最后一行使用 --boundary-- 结尾
