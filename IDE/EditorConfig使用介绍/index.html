EditorConfig使用介绍

发表于 2018-12-11 更新于 2023-08-09 分类于 IDE

在github看到很多项目根目录都有 .editorconfig 文件，所以去了解一下。

这个文件是 EditorConfig 用来统一不同编辑器的代码风格的配置。

各种IDE的风格自然不一样，通过这样一个配置文件定义的标准统一风格，比如缩进用x个空格。再有就是多人一起协同开发一个项目，editorconfig 能起到很好的效果。
