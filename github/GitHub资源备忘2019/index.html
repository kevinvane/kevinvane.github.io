TCP proxy in ANSI C

Github

这是一个简单的TCP代理转发程序，允许您将访问本地主机上指定端口的TCP请求转发到另一台主机上的不同端口。它还可以通过外部命令转发数据(用于记录、过滤或复制网络流量)。它是用ANSI C编写的，所以占用的空间很小，可以用于嵌入式设备。

注: 大概就是和nginx的反向代理功能差不多，它的优势是小，可以在嵌入式设备上跑。
