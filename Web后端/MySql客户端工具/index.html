MySql客户端工具

发表于 2023-07-10 更新于 2023-08-09 分类于 Web后端

记录一下MySQL的客户端连接工具软件

一、商业&收费

  1. Navicat
  2. SQLyog
  3. DataGrip

收费的软件不用说，每个都是很强大，适合专业人士。

二、开源&免费

收费软件是很强大，但我们有时候只是需要简单操作一下数据库，收费软件的很多功能是用不到的，比如：设计复杂的数据库等。那么可以考虑支持一波开源软件，特别是轻量级的软件，很适合。

  1. DBeaver社区版
    用Java编写的一个免费的跨平台数据库工具，界面GUI库是基于Eclipse。它支持所有流行的SQL数据库，如MySQL, MariaDB, PostgreSQL, SQLite, Apache Family等，社区版不支持NoSQL，收费版才支持NoSQL。功能强大。
    推荐，理由：支持数据库类型很多，几乎都支持了，而且功能和收费软件比一样不少。

  2. phpMyAdmin
    用PHP编写的SQL客户端，通过web方式控制和操作MySQL数据库。
    不推荐，理由：安装部署不方便，界面有点老（丑），用不习惯。

  3. HeidiSQL

    用Delphi 编写的SQL轻量级客户端，Github 。支持 MySQL/MariaDB、PostgreSQL、SQLite、SQL Server、Interbase 和 Firebird ，仅支持Windows，在Linux下依赖Wine。
    推荐，理由：轻量级，界面清爽，该有的功能都有。




  4. Antares SQL
    基于Electron.js + Vue.js的开源轻量级SQL 客户端，支持 MySQL/MariaDB, PostgreSQL 和 SQLite。
    推荐，理由：轻量级，支持数据导入导出，足够使用，而且还支持跨平台。

    安装提示：可以去Github下载安装包，Antares-0.7.0-portable.exe 大概70MB，无需安装，exe直接运行，但更方便的安装方式是在商店安装，因为它已上架 Snap Store 和 微软商店。
