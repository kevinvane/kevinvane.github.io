简介

【sphinx-doc】是基于python的文档生成器（使用文档推荐【这个】），它使用 reStructuredText 语法(类似markdown)编写文档，通过编译，可以生成html,epub,man,pdf等多种格式，现在也可以通过安装扩展来支持简单的Markdown语法。
