安装 AlmaLinux9.0 DVD和预览

发表于 2022-11-29 分类于 linux

CentOS 8 在2021年12月31日停止更新，取而代之的是 CentOS Stream （RHEL的测试版）。此情况下诞生了其他的发行版本 CentOS，比如：

  1. AlmaLinux 由 CloudLinux OS 的创建者打造，已在业界取得了一定的知名度。
  2. Rocky Linux 由卖掉 CentOS 的 Gregory Kurtzer 发起，由社区驱动。

** AlmaLinux 跟 RHEL 完全在二进制上兼容，承诺永久免费。**
